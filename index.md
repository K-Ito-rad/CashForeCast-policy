# CashForeCast プライバシーポリシー

最終更新日：2026-02-23

## 1. 収集する情報
本アプリは、以下の情報を収集する場合があります。

- **クラッシュデータ**：アプリがクラッシュした際に生成される技術情報（安定性改善のため）
- **匿名の利用状況データ（任意）**：機能の利用状況など、個人を特定しない形での統計情報（改善のため）

※本アプリは、氏名・住所・電話番号・メールアドレス・連絡先・写真・位置情報などの個人情報を収集しません。

## 2. 利用目的
収集した情報は、以下の目的のみに利用します。

- 不具合の調査・修正
- 安定性およびユーザー体験の改善

## 3. 第三者提供
本アプリは、取得した情報を第三者へ販売しません。
また、広告目的で第三者に提供しません。

## 4. データの保存
本アプリに入力された内容（例：収入・支出・メモ等）は、原則として**端末内に保存**されます。
（将来的にクラウド同期等の機能を追加する場合は、別途明示します。）
